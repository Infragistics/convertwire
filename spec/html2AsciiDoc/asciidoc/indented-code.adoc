1.  Add a namespace declaration for xamDataGrid.

*In XAML:*

----
xmlns:igDP="http://infragistics.com/DataPresenter"
----

*In Visual Basic:*

[source,vb]
----
Imports Infragistics.Windows.DataPresenter
----

*In C#:*

[source,csharp]
----
using Infragistics.Windows.DataPresenter;
----
